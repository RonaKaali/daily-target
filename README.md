# DailyTarget | Professional SaaS Planner 🎯

DailyTarget is a modern, gamified Calendar & Daily Target Planner designed to boost productivity. It combines beautiful, responsive design with interactive planning tools to help users track their daily goals, maintain habits, and measure their progress effectively.

## 🌟 Key Features

- **Daily Target Tracking**: Set, manage, and track your daily tasks and targets efficiently.
- **Gamification (XP Progress)**: Earn XP and level up as you complete tasks. Stay motivated with a visual progress system.
- **Interactive Calendar**: Plan your month ahead with the built-in, responsive calendar.
- **Focus Mode**: Minimize distractions and focus on your most critical tasks.
- **Modern & Responsive UI**: Built with elegant design paradigms like glassmorphism, smooth animations, and a rich dark mode.

## 💻 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Data Visualization**: [Chart.js](https://www.chartjs.org/) & [react-chartjs-2](https://react-chartjs-2.js.org/)
- **Date Utilities**: [date-fns](https://date-fns.org/) & [react-day-picker](https://daypicker.dev/)
- **Fonts**: Outfit & Plus Jakarta Sans (Google Fonts optimization)

## 🚀 Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites
Make sure you have Node.js and npm (or pnpm/yarn/bun) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/daily-target.git
   ```
2. Navigate to the project directory:
   ```bash
   cd daily-target
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) (or the port specified in your terminal) with your browser to see the application.
